Cara login sulingjar

login untuk guru mengisi survei sulingjar (survei lingkungan belajar) dapat mengunjungi link https://surveilingkunganbelajar.kemdikbud.go.id/ atau klik disini  

setelah itu anda akan diminta memasukkan

  • NPSN
  • Token
  • NIK
  • Tanggal lahir

Mendapatkan token sulingjar

bagaimana mendapatkan token dan akun sulingjar?

operator dapat mengunduh akun sulingjar guru melalui tautan ini https://dashboardslb.kemdikbud.go.id/ atau klik disini

Mengisi survei sulingjar

ada beberapa keynote tentang sulingjar:

  1. ada 45 soal survei sulingjar
  2. soal ini bersifat survei, jangan takut salah menjawab. isilah sesuai dengan kondisi lingkungan sekolah anda, karena akan membantu meningkatkan kualitas pendidikan di Indonesia.
  3. disarankan menjawab menggunakan perangkat laptop
  4. soal otomatis tersimpan, namun disarankan menyelesaikan langsung soal karena sistem bisa mereset ulang (ketika anda login dengan perangkat lain)
  5. disarankan mengerjakan saat malam hari karena server menjadi lambat ketika dibuka siang hari dimana banyak user lain
