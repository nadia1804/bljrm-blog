Langkah 1: Verifikasi Domain di Bing Webmaster

  1. Buka Bing Webmaster Tools
  2. Tambahkan dan verifikasi domain kamu
  3. Setelah domain terverifikasi, lanjut ke langkah berikutnya

Langkah 2: Generate API Key IndexNow

  1. Buka halaman IndexNow Get Started
  2. Klik tombol Generate API Key
  3. Unduh file `.txt` yang berisi key tersebut

klik download pada Generate API Key, Klik tombol generate jika ingin mengganti API Key, abaikan point nomor 2 pada gambar

File yang didownload berupa ApiKey.txt seperti ini Contoh:

a230e4ddd7dd480193bc9f2aebc4e162.txt

Isi Apikey.txt seperti ini:
a230e4ddd7dd480193bc9f2aebc4e162

Langkah 3: Upload File `.txt` ke Root Domain

Upload file `.txt` ke root domain kamu, misalnya:
https://domainkamu.com/a230e4ddd7dd480193bc9f2aebc4e162.txt

Apakah bisa untuk blogspot? sayangnya belum bisa. indexnow khusus website yang memiliki akses ke server

Langkah 4: Verifikasi File `.txt`

Cek file di browser:
https://domainkamu.com/a230e4ddd7dd480193bc9f2aebc4e162.txt

jika berhasil maka muncul tulisan di browser :

dan Cek Ping manual via terminal:
curl.exe -i "https://api.indexnow.org/indexnow?url=https://domainkamu.com/&key=a230e4ddd7dd480193bc9f2aebc4e162"

agar lebih mudah melakukan IndexNow kita akan lakukan secara otomatis, kita perlu membuat 2 buah file bernama :

  1. indexnow.bat
  2. indexnow-data.txt

indexnow.bat berfungsi sebagai script melakukan ping otomatis, sedangkan indexnow-data.txt berfungsi sebagai database sitemap domain yang akan di ping.

Buat File 'indexnow-data.txt'

buat file dengan nama indexnow-data.txt dengan isinya adalah daftar sitemap kamu tanpa dot (titik) :

  • https://domainkamu.com/post1
  • https://domainkamu.com/post2
  • https://domainkamu.com/post3

Langkah 6: Jalankan Skrip

  • Klik dua kali `indexnow.bat`
  • Biarkan skrip berjalan dan mengirim ping ke IndexNow
  • Jika berhasil, URL kamu akan mulai muncul di dashboard Bing Webmaster Tools
